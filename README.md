# 🔐 Secure Password Generator

A modern **Password Generator desktop application** built using **Python** and **CustomTkinter**.
This tool generates **strong, random, and secure passwords** using cryptographically secure methods.

The application features a **modern dark UI**, **password strength indicator**, and **copy-to-clipboard functionality**, making it both practical and visually appealing.

---

## 🚀 Features

* 🔑 Generate **secure random passwords**
* 🔢 Adjustable **password length (6–32 characters)**
* 🔐 Uses Python's **`secrets` module** for cryptographically secure randomness
* 📋 **Copy password to clipboard** with one click
* 📊 **Password strength indicator with progress bar**
* 🎨 Modern **dark theme UI**
* ⚡ Smooth and responsive interface

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/pallavi-676/random-password-generator.git
```

2. Navigate into the project folder

```
cd random-password-generator
```

3. Install required dependency

```
pip install customtkinter
```

## 🔒 How It Works

1. Choose the desired **password length** using the slider.
2. Click **Generate** to create a strong password.
3. The password will appear in the display box.
4. Use **Copy** to copy it to your clipboard.
5. The **strength indicator** shows whether the password is weak, medium, or strong.
